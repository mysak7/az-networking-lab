Optional Extension — Private Endpoint

Commented out in main.tf

Private DNS + Blob Private Endpoint

Uncomment the private endpoint block in main.tf to provision a private IP for the storage account inside the subnet. After applying, SSH into the VM and run nslookup <storage>.blob.core.windows.net — it resolves to the private IP instead of the public Azure endpoint, demonstrating private DNS resolution over the VNet link.

azurerm_private_dns_zone azurerm_private_dns_zone_virtual_network_link azurerm_private_endpoint
