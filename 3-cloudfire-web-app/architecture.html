Setup Checklist

  1. 1Add mysak.fun to Cloudflare (Free plan) — get 2 nameservers
  2. 2Update NS records in Azure DNS Zone → Cloudflare nameservers
  3. 3Copy Zone ID from Cloudflare dashboard → terraform.tfvars
  4. 4Run terraform apply — CNAME + TXT records created automatically
  5. 5App Service custom domain bound via TXT verification (works with orange cloud)
  6. 6Deploy zips landing.html and uploads via az webapp deploy
